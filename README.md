# Ripple-Cli-Rake

Extension for ripple-cli gem. Wraps ruby classes around the ripple command line parameters.

## Installation

Add this line to your application's Gemfile:

    gem 'ripple-cli-rake'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ripple-cli-rake
